# 🌾 Smart Farm-to-Market Supply Chain Optimization using Power BI
Smart Farm-to-Market Supply Chain Optimization using Power BI for analyzing post-harvest losses, warehouse efficiency, transportation impact, market demand, revenue trends, and supply chain risk through interactive dashboards and KPI analysis.

##  Project Overview
This project focuses on analyzing and optimizing the agricultural supply chain using Power BI dashboards and data analytics techniques.

The main objective of this project is to identify inefficiencies in transportation, storage, market demand, insurance coverage, and post-harvest handling to reduce crop losses and improve operational efficiency.

The dashboard provides interactive visualizations and KPI analysis to support data-driven decision-making in the agriculture supply chain sector.

---

#  Key Objectives

- Analyze post-harvest losses
- Monitor transportation efficiency
- Evaluate warehouse utilization
- Analyze market demand trends
- Identify revenue leakage
- Improve supply chain visibility
- Support better decision-making using data analytics
